    Alcohol detection and Accident prevention system using Arduino

    Alcohol detection in vehicles which an Alcohol Sensor(MQ135) detects alcohol consumption by smell of alcohol from driver. Whenever an inebriated driver starts the vehicle, the vehicle will not start. And if driver gets into inebriated condition while driving the vehicle will come to rest, start indication lights and sends details to police authorities. By which we can reduce/prevent the Drunk and Drive accidents.

    Fire detection and Accident prevention system using Arduino

    Fire detection and prevention in vehicles is the system in which Fire sensor detects fire and starts water sprinkler/fire extinguisher, which is an actuator and stop only when the fire is under control. The system also opens up the windows and emergency exits and informs the fire department.
